##<a name="basic"/>基本用法
	window.easyScroll( y , steps );
    
######如果不想设置scrollWrap的的宽度，请给scrollWrap设置display: inline-flex;属性方可自动撑开宽度而不换行。


##<a name="options"/>参数详解
#####y [Number] 页面需要滚到到的Y轴位置。
#####steps [Number] 定义此次滚动的总帧数。默认值为30。
